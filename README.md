# 🐍 Proyecto de Python - Katas

## 📌 Descripción

Este proyecto consiste en la resolución de diferentes **katas en Python** con el objetivo de practicar y demostrar los conocimientos adquiridos durante el curso.

Las katas abordan distintos problemas y están diseñadas para reforzar habilidades clave en programación.

---

## 🎯 Objetivos

A lo largo del proyecto se busca demostrar el manejo de:

- Tipos de datos básicos y funciones incorporadas  
- Estructuras de datos en Python (listas, tuplas, diccionarios, etc.)  
- Condicionales (`if`, `elif`, `else`)  
- Estructuras de iteración (`for`, `while`)  
- Funciones en Python  
- Clases y Programación Orientada a Objetos (POO)  
- Uso de módulos y librerías  
- Buenas prácticas de programación  

---

## 🧠 Contenidos trabajados

### 🔹 Tipos de datos
Uso de enteros, floats, strings y booleanos.

### 🔹 Estructuras de datos
Manipulación de:
- Listas  
- Tuplas  
- Diccionarios
- sets  

### 🔹 Control de flujo
- Condicionales  
- Bucles  

### 🔹 Funciones
Creación y uso de funciones para modularizar el código.

### 🔹 Programación Orientada a Objetos
- Definición de clases  
- Métodos
